Console commands:
as [file].asm - assemble
load [file].obj - loads
list [mem] - goes to specified line
break set [mem] - breaks program at specified line
script [file] - runs script that just uses the console thing

Script functions:
loadOS - assembles and loads os into mem
run - assembles and load current project

starting registers:
PC = x0200
MPR = x0000
PSR = x8002
CC = Z
