Console commands:
as [file].asm - assemble
load [file].obj - loads
list [mem] - goes to specified line and reads out contained data
break set [mem] - breaks program at specified line
script [file] - runs script that just uses the console thing
reset - resets simulator
help - helps you

Script functions:
os - assembles and loads os into mem (clears output console too)
run - assembles and load current project

Basic Colors (RGB):
x7FFF	White
x0000	Black
x7C00	Red
x03E0	Green
x001F	Blue
x3466	Puce
